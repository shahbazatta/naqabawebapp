22. List of Unknown IMEIs and port number

Return List of IMEIs and port number without details during a given time period, to access this api required api key or token.

Access Url

To get characters you need to make a GET call to the following url :
https://tracking.naqaba.com.sa/api/getStatusUnknown?token=cebc8011932a85c60a7e079b840bf083161812d3


QUERY PARAMETERS

Field Type Description
token String cebc8011932a85c60a7e079b840bf083161812d3

Response


{
	"status": 200,
	"result": "success",
	"count": 2,
	"data": [{
		"imei": 350544504943914,
		"port": 3001
	}, {
		"imei": 863540062349916,
		"port": 3001
	}]
}


PARAMETERS

Field Type Description
count Integer 2
imei Integer 350544504943914
port Integer 3001
Error Code Meaning
404 Bad Request : Invalid argument (invalid request payload)
403 Forbidden : Permission denied (e.g. invalid token).
