17. List of all buses completed at least 1 round

Return the list of all buses that have completed atleast 1 round and total number of rounds with imei during a given time period and geofence, to access this api required api key or token.

Access Url

To get characters you need to make a GET call to the following url :
https://tracking.naqaba.com.sa/api/getAtleastOneRoundListTimePeriod?src_geo=P5&dest_geo=S5&start_dt=15-04-2023&start_tm=16:00:00&end_dt=15-04-2023&end_tm=23:00:00&token=cebc8011932a85c60a7e079b840bf083161812d3


QUERY PARAMETERS

Field Type Description
token String cebc8011932a85c60a7e079b840bf083161812d3
src_geo String P2
dest_geo String S5
Start Date(start_dt) String 01-04-2023
Start time(start_tm) String 00:00
End Date(end_dt) String 01-04-2023
End time(end_tm) String 00:00
