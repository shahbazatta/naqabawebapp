11. Bus Location History(API 14)

This API will Description Return the location of the bus in a given time period, to access this api required api key or token.

Access Url

To get characters you need to make a GET call to the following url :
https://tracking.naqaba.com.sa/api/getLocationBusTimePeriod?imei=867698043936692&start_dt=11-04-2023&start_tm=00:00:00&end_dt=11-04-2023&end_tm=23:59:59&token=cebc8011932a85c60a7e079b840bf083161812d3


QUERY PARAMETERS

Field Type Description
token String cebc8011932a85c60a7e079b840bf083161812d3
imei Integer 358480083206997
Start Date(start_dt) String 01-04-2023
Start time(start_tm) String 00:00
End Date(end_dt) String 01-04-2023
End time(end_tm) String 00:00
