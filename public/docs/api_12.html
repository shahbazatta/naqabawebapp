12. All Round Trips (A-B-A)(API 4)

Return the list all round trips (A-B-A) with total time taken for each round for all buses between Source and Destination during a given time period , to access this api required api key or token.

Access Url

To get characters you need to make a GET call to the following url :
https://tracking.naqaba.com.sa/api/getRTTforAllBussesTimePeriod?src_geo=P5&dest_geo=S5&start_dt=11-04-2023&start_tm=12:00:00&end_dt=11-04-2023&end_tm=13:03:00&token=cebc8011932a85c60a7e079b840bf083161812d3


QUERY PARAMETERS

Field Type Description
token String cebc8011932a85c60a7e079b840bf083161812d3
src_geo(source point geo code) String P5
dest_geo(destination point geo code) String S5
Start Date(start_dt) String 01-04-2023
Start time(start_tm) String 00:00
End Date(end_dt) String 01-04-2023
End time(end_tm) String 00:00
