1. All AVL Live Locations

This API provides list of all AVL Devices last live location, to access this api no api key or no token is required.

Access Url

To get characters you need to make a GET call to the following url :
http://tracking.naqaba.com.sa/api/getDevicesDataLive?token=cebc8011932a85c60a7e079b840bf083161812d3&min=10
