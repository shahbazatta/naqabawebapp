14. Total Round Trip Count

Return the total number of round trips between all GeoFences based on given day or with day and time, to access this api required api key or token.

Access Url

To get characters you need to make a GET call to the following url :
https://tracking.naqaba.com.sa/api/getTTCountTimePeriod?start_dt=11-04-2023&start_tm=12:00:00&end_dt=11-04-2023&end_tm=13:03:00&token=cebc8011932a85c60a7e079b840bf083161812d3


QUERY PARAMETERS

Field Type Description
token String cebc8011932a85c60a7e079b840bf083161812d3
Start Date(start_dt) String 01-04-2023
Start time(start_tm) String 00:00
End Date(end_dt) String 01-04-2023
End time(end_tm) String 00:00

Response


{
	"TotalRoundTrip": 85,
	"data": [{
		"src_geo": "P1",
		"dest_geo": "P11",
		"RoundTrip": 38
	}, {
		"src_geo": "P2",
		"dest_geo": "P11",
		"RoundTrip": 9
	}......]
}


PARAMETERS

Field Type Description
TotalRoundTrip Integer 85
Error Code Meaning
