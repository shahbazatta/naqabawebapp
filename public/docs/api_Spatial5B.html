19. Average time of the trip (A-B and B-A)

Return the average time of the trip (A-B and B-A) for each IMEI during a given time period and geofence, to access this api required api key or token.

Access Url

To get characters you need to make a GET call to the following url :
https://tracking.naqaba.com.sa/api/getAvgTTAllBussesTimePeriod?src_geo=P5&dest_geo=S5&start_dt=15-04-2023&start_tm=16:00:00&end_dt=15-04-2023&end_tm=23:00:00&token=cebc8011932a85c60a7e079b840bf083161812d3
