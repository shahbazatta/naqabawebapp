15. Average time of consecutive bus leaving

Return the average time of consecutive bus leaving from the given geofence based on a given time, to access this api required api key or token.

Access Url

To get characters you need to make a GET call to the following url :
https://tracking.naqaba.com.sa/api/getAvgTimeBusLeavingTimePeriod?src_geo=P2&start_dt=11-04-2023&start_tm=12:00:00&end_dt=11-04-2023&end_tm=13:03:00&token=cebc8011932a85c60a7e079b840bf083161812d3


QUERY PARAMETERS

Field Type Description
token String cebc8011932a85c60a7e079b840bf083161812d3
src_geo String P2
Start Date(start_dt) String 01-04-2023
Start time(start_tm) String 00:00
End Date(end_dt) String 01-04-2023
End time(end_tm) String 00:00
