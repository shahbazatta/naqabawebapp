7. Return the list of buses available in a given geofence and time period

Return the list of buses available in a given geofence and time period. Bus will be counted if any of their location is in the given geofence in a given time period, to access this api required api key or token.

Access Url

To get characters you need to make a GET call to the following url :
https://tracking.naqaba.com.sa/api/getListBusesGeofenceTimePeriod?geocode=S1&start_dt=01-04-2023&start_tm=12:18:00&end_dt=03-04-2023&end_tm=23:18:00&token=cebc8011932a85c60a7e079b840bf083161812d3


QUERY PARAMETERS

Field Type Description
token String cebc8011932a85c60a7e079b840bf083161812d3
geocode String S1
Start Date(start_dt) String 01-04-2023
Start time(start_tm) String 00:00
End Date(end_dt) String 01-04-2023
End time(end_tm) String 00:00
