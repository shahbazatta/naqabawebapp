13. Two AVL Reading Time difference

Return the list of the buses where the difference between any two consecutive readings more than x minutes in a given time period, to access this api required api key or token.

Access Url

To get characters you need to make a GET call to the following url :
https://tracking.naqaba.com.sa/api/getDelayBetweenReadingsTimePeriod?imei=867698044173543&min=5&start_dt=15-04-2023&start_tm=00:00:00&end_dt=15-04-2023&end_tm=10:21:59&token=cebc8011932a85c60a7e079b840bf083161812d3


QUERY PARAMETERS

Field Type Description
token String cebc8011932a85c60a7e079b840bf083161812d3
imei Integer 862095054357147 (Unique ID of AVL Devices 15 digits Number)
min Integer 5(minutes)
Start Date(start_dt) String 01-04-2023
Start time(start_tm) String 00:00
End Date(end_dt) String 01-04-2023
End time(end_tm) String 00:00
