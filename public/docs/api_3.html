3. All AVL Data Not Received in x Minutes

This API provides list of all AVL Devices not received live location within x minutes, to access this api required api key or token.

Access Url

To get characters you need to make a GET call to the following url :
https://tracking.naqaba.com.sa/api/getDevicesNotSynced?min=600&token=cebc8011932a85c60a7e079b840bf083161812d3


QUERY PARAMETERS

Field Type Description
token String cebc8011932a85c60a7e079b840bf083161812d3
min Integer 10(In minutes)
