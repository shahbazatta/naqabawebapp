6. Bus is availble on given locations

Return whether bus is available in a given location or not., to access this api required api key or token.

Access Url

To get characters you need to make a GET call to the following url :
https://tracking.naqaba.com.sa/api/getBusStatus?plate_no=%D8%A3%20%D8%B1%20%D8%B5%206464&lng=39.94312286376953&accuracy=50000&min=11&lat=21.242998123168945&token=cebc8011932a85c60a7e079b840bf083161812d3
