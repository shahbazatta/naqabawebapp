12. Two AVL Reading Time difference - List of IMEIs

This API Return List of All IMEIs, to access this api required api key or token.

Access Url

To get characters you need to make a GET call to the following url :
https://tracking.naqaba.com.sa/api/getAllImeis?token=cebc8011932a85c60a7e079b840bf083161812d3
OR
https://tracking.naqaba.com.sa/api/getAllImeis?port=3004&token=cebc8011932a85c60a7e079b840bf083161812d3

token String cebc8011932a85c60a7e079b840bf083161812d3
